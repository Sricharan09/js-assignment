Different Console Methods

Press F12 on your keyboard to view the message in the console view.

This console.log() - Outputs a message to the console

This console.error() - Outputs an error message to the console

This console.warn() - Outputs a warning message to the console

This console.info() - Outputs an informational message to the console

This console.time() & console.timeEnd - starts the timer and Stops a timer that was previously started by console.time()

This console.count() - Logs the number of times that this particular call to count() has been called

This console.assert() -This method writes a message to the console, but only if an expression evaluates to false

This console.trace() - method displays a trace that show how the code ended up at a certain point
