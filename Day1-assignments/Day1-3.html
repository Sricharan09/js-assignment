Javascript Data Types

JavaScript Strings
  1. A string (or a text string) is a series of characters like "John Doe".
  2. Strings are written with quotes.
  3. You can use single or double quotes.
JavaScript Numbers
  1. JavaScript has only one type of numbers.
  2. Numbers can be written with, or without decimals.
JavaScript Booleans
    Booleans can only have two values: true or false.
JavaScript Arrays
  1. JavaScript arrays are written with square brackets [].
  2. Array items are separated by commas.
JavaScript Objects
  1. JavaScript objects are written with curly braces {}.
  2. Object properties are written as name:value pairs, separated by commas.
Undefined
  1. In JavaScript, a variable without a value, has the value undefined.
  2. The type is also undefined.
Null
  1. In JavaScript null is "nothing".
  2. It is supposed to be something that doesn't exist.
  3. Unfortunately, in JavaScript, the data type of null is an object.
